# LLM_MultiAgent_Applications 目录说明

## 组别概述
聚焦于利用大型语言模型（LLM）赋能的多智能体系统在教育场景中的具体应用与实证研究，如课堂模拟、复杂问题解决等。

## 文件详细索引 (供 AI 精准调取)
### 文件名：`利用LLM赋能的智能体模拟课堂教学.jsonl`
**核心内容**：1. 核心研究领域：利用大型语言模型（LLM）赋能的多智能体系统进行虚拟课堂教学模拟。
2. 关键技术：提出SimClass多智能体框架，包含课堂角色识别与课堂控制机制，并基于弗兰德斯交互分析系统（FIAS）和探究共同体理论进行评估。
3. 核心结论：LLM能够模拟出支持师生及生生积极互动的动态学习环境，智能体通过协作创造生动课堂互动以改善用户学习过程。

### 文件名：`基于多智能体的人机协同解决复杂学习问题实证研究.jsonl`
**核心内容**：核心研究领域为人机协同解决复杂学习问题。关键技术为基于大语言模型的多智能体框架（如AutoGen）。核心结论是，相比单智能体，多智能体环境能促使学习者自发运用多维提问策略，更高效地解决复杂问题，并揭示了单体对话模式的局限性。
