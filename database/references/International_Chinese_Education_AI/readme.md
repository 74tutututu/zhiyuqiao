# International_Chinese_Education_AI 目录说明

## 组别概述
探讨以ChatGPT为代表的生成式人工智能及数智技术在国际中文教育中的应用、带来的机遇、挑战及教学变革。

## 文件详细索引 (供 AI 精准调取)
### 文件名：`“ChatGPT来了：国际中文教育的新机遇与新挑战”大家谈 (1).jsonl`
**核心内容**：核心研究领域：ChatGPT等生成式人工智能在国际中文教育中的应用与影响。
关键技术：基于大语言模型的自然语言处理、人机对话、智能测评、个性化学习路径推荐。
核心结论：ChatGPT为国际中文教育带来个性化教学、智能测评等新机遇，但也面临内容准确性、文化理解、伦理及教师角色重塑等挑战。需推动“技术+教育”深度融合，构建人机协同的智慧教育新生态。

### 文件名：`“ChatGPT来了：国际中文教育的新机遇与新挑战”大家谈.jsonl`
**核心内容**：核心研究领域为国际中文教育在人工智能时代的发展。关键技术涉及以ChatGPT为代表的大语言模型、生成式AI及知识图谱构建。核心结论是：AI为教学资源建设、个性化学习、教师角色转变带来机遇，但存在内容准确性、伦理风险等挑战；需推动人机协同，构建智慧教育新生态。

### 文件名：`曹贤文等. “数智时代的国际中文教育：变与不变”大家谈.jsonl`
**核心内容**：核心研究领域：数智时代国际中文教育的变革与创新。
关键技术/方法论：人机协同教学、大语言模型（LLM）、AIGC、虚拟仿真/元宇宙、多模态融媒词典、大数据循证评价。
核心结论：数智技术深刻改变教育生态，需构建教师数智能力框架，推动教材、词典、教学模式向资源化、个性化、互动化转型，但教师的核心主导地位和人文关怀不可替代。
