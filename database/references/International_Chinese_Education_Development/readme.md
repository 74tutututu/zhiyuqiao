# International_Chinese_Education_Development 目录说明

## 组别概述
关注国际中文教育的宏观发展战略、转型路径、社会支持及在促进文明互鉴中的角色，不局限于技术应用。

## 文件详细索引 (供 AI 精准调取)
### 文件名：`搭建语言互通之桥 加强文明互信互鉴——习近平主席致2024世界中文大会的贺信为开展好国际中文教育凝聚共识.jsonl`
**核心内容**：1. 核心研究领域：国际中文教育及其在促进文明互鉴和构建人类命运共同体中的作用。
2. 关键技术/方法论：通过孔子学院、中文纳入国民教育体系、创新教学方法及“汉语桥”等平台，开展多层次、精准化的语言文化教学与交流。
3. 核心结论：中文是重要的公共文化产品，国际中文教育广受欢迎，是增进理解、促进民心相通的关键桥梁，需凝聚共识以开创教育新局面。

### 文件名：`新时代国际中文教育的转型向度、现实挑战及因应对策.jsonl`
**核心内容**：核心研究领域：新时代国际中文教育的转型发展。
关键技术/方法论：提出转型向度（使命、方式、重心、人才培养转变），分析“五关”挑战（内涵、人才、平衡、安全、可持续发展），并构建对策体系。
核心结论：国际中文教育需从规模扩张转向内涵式高质量发展，通过优化全球布局、加强标准建设、推进人才培养多元化及本土化、特色化、智慧化发展，以应对现实挑战，构建开放包容的现代教育体系。

### 文件名：`智慧教育背景下国际中文学习者在线学习社会支持研究.jsonl`
**核心内容**：核心研究领域为智慧教育背景下国际中文学习者的在线学习社会支持。研究采用问卷调查法，使用修订的领悟社会支持量表和学习投入量表，并结合访谈。核心结论是：在线学习共同体支持和家人支持与学习投入呈显著正相关，而朋友支持相关性不显著；在线学习共同体的支持水平最低，尤其缺乏同伴互动与情感支持。建议加强在线同伴互动、提升教师情感支持，并促进线上线下社区融合。
