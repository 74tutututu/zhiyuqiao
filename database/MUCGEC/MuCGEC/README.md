# MuCGEC 国际中文教育语法纠错平行语料库

## 核心教育价值：一错多改 (Multi-Reference)
在真实的国际中文教学中，一个语病往往有多种地道的修改方式。本数据集的最大价值在于提供了“多参考标准答案”，帮助 AI 智能体掌握灵活、自然的中文表达习惯，而非单一的机械纠错。

## 数据集结构说明

### 1. 专家批改范例集 (`MuCGEC_dev.txt`)
包含了超过 1000 句带有详细修改方案的留学生病句。大模型可以通过该文件学习如何对同一个错误进行多维度修正。
