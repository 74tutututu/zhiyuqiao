# MUCGEC 国际中文语法纠错总目录

本目录是国际中文教育领域语法纠错（GEC）的核心资源集合，旨在为大模型提供从规范标准到实际案例的完整知识体系，以支持高级、灵活的汉语作文批改与偏误诊断能力。
