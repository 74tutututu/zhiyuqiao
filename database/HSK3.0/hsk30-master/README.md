# HSK 3.0 中文水平等级标准数据库 (HSK 3.0 Grading Standards)

本目录是智能体的“难度控制与评估游标尺”，收录了最新版 HSK 3.0（三等九级）的全部底层数据。由于数据已高度结构化为 CSV 格式，大模型可以直接精确读取并匹配字段。

## 核心数据表清单
1. **`hsk30.csv` (核心词汇表)**：包含各等级必须掌握的核心词汇。
2. **`hsk30-expanded.csv` (扩展词汇表)**：包含在核心词汇基础上的衍生词与补充高频词汇。
3. **`hsk30-chars.csv` (核心汉字表)**：包含各等级汉字的字级、拼音，用于指导汉字的认读与书写教学。
4. **`hsk30-grammar.csv` (核心语法表)**：包含初、中、高三个等次及九个级别的语法结构与句型。

## 词汇表数据字典 (Data Schema)
在解析 `hsk30.csv` 和 `hsk30-expanded.csv` 时，请严格参考以下字段定义：
* **`ID`**: 唯一键，格式为 `Ln-nnnn`（级别 + 原始序号）。7-9级统一带有 `L7` 前缀。
* **`Simplified`**: 简体中文词汇（依据 HSK 官方标准）。
* **`Traditional`**: 对应的繁体中文词汇，若有异体字则以 `|` 分隔。
* **`Pinyin`**: 包含声调符号的规范拼音。
* **`Level`**: 词汇所属的 HSK 3.0 等级，值为 `1`, `2`, `3`, `4`, `5`, `6` 或高级别的 `"7-9"`。
* **`POS`**: 词性（Part of Speech），包含多个词性时以 `/` 分隔。英文缩写与中文词性映射如下：
  * `N` (名): 名词 (Noun)
  * `V` (动): 动词 (Verb)
  * `Adj` (形): 形容词 (Adjective)
  * `Adv` (副): 副词 (Adverb)
  * `Pron` (代): 代词 (Pronoun)
  * `Num` (数): 数词 (Numeral)
  * `M` (量): 量词 (Measure word / Classifier)
  * `Prep` (介): 介词 (Preposition)
  * `Conj` (连): 连词 (Conjunction)
  * `Aux` (助): 助词 (Auxiliary word / Particle)
  * `Int` (叹): 叹词 (Interjection / Exclamation)
