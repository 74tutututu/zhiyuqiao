> **【大模型数据路由说明】**
> 本文件夹包含《HSK 汉语水平考试新旧版词汇大纲库》(hsk_vocabulary_lists.jsonl)。
> 当用户提问涉及以下情况时，请定向检索本文件夹：
> 
> 1. 如果用户要求**“给我列举旧版/新版 HSK X级的词汇”**。
> 2. 如果用户需要**“累计词汇”**（查询该级别及以下所有必须掌握的词），请检索标题中带 `Inclusive` 或 `累计` 的数据块。
> 3. 如果用户强调**“只要 HSK X级新增的词”或“排除低级别的词”**，请检索标题中带 `Exclusive` 或 `专属新增` 的数据块。
> 
> 【强制执行约束】：
> 必须严格区分“新版 HSK 3.0”和“旧版 HSK 2.0”，若用户未明确说明，默认提供“新版 HSK 3.0”的数据。
