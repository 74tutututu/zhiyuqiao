# Educational_Agents_Technology 目录说明

## 组别概述
专注于教育智能体（Pedagogical Agents）的技术发展、系统架构与应用研究，涵盖情感交互、多智能体协作、大模型驱动等关键技术。

## 文件详细索引 (供 AI 精准调取)
### 文件名：`Emotion-Related-Pedagogical-Agent.jsonl`
**核心内容**：1. 核心研究领域：探究教学代理如何满足学习者的情感需求，特别是在数字学习环境中。
2. 关键技术/方法：采用Kitchenham方法对2013-2023年间Scopus Q1-Q4期刊文献进行系统综述，分析模型、干预类型与输入方式。
3. 核心结论：现有教学代理缺乏针对学习者情感状态进行个性化干预的专用模型，多数干预适应性不足且应用于异步学习（如电子学习）；极少有代理利用人工智能实时识别情感以触发自适应干预。

### 文件名：`教育智能体的发展历程、应用现状与未来展望.jsonl`
**核心内容**：1. 核心研究领域：教育智能体（Pedagogical Agents），即人工智能与教育融合的虚拟角色，旨在实现教育的智能化、精准化和个性化。
2. 关键技术：智能体技术、大数据、机器学习、情感计算、深度学习，应用于智能导师系统、虚拟角色建模与人机交互。
3. 核心结论：教育智能体在支持个性化学习、扮演教学角色和初步情感交互方面已有应用，但存在决策精度弱、对话能力有限、情感交互不足等问题。未来需与大数据、深度学习等技术深度融合，以实现更深层的情感交互和精准教学决策。

### 文件名：`基于大模型的教学智能体构建与应用研究.jsonl`
**核心内容**：核心研究领域为基于大模型的教学智能体构建及其在教育（特别是项目式学习）中的应用。关键技术包括以大模型为核心，整合检索增强生成、思维链/ReAct等推理规划方法，并利用LangChain、AutoGen等框架实现多模态感知、任务规划与多智能体协作。核心结论是提出了一个包含任务设定、规划、能力实现、记忆反思和交互进化模块的教学智能体框架，论证了其作为“助教”和“同伴”在项目式学习全流程中的支持作用，并指出了未来需发展专用教育大模型及关注伦理。

### 文件名：`大模型驱动的教育多智能体系统应用研究——技术架构、发展现状、实践路径与.jsonl`
**核心内容**：核心研究领域：大模型驱动的教育多智能体系统，旨在通过多角色智能体协作解决复杂教育任务。
关键技术：构建了包含单智能体层、多智能体系统层、场景感知层和评估层的四层技术架构，涉及智能体画像、任务规划、记忆管理、协作编排等关键技术。
核心结论：该系统在教学辅助、学习支持、评价与研究等方面展现出优势，并提出了“机制设计-系统建模-效果评估”的闭环实践路径，但其发展仍面临透明度、可靠性、评估验证等挑战。
