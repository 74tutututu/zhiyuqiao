# 多语种母语者偏误与对比教学策略库 (Contrastive Teaching Strategies)

本目录是智能体的“国别化教学大脑”，专门收录了基于对比语言学（Contrastive Linguistics）的国际中文教学实战策略。通过对比不同语言与汉语在语音、词汇、语法系统上的差异，提前预判学生的偏误并提供解决方案。

## 涵盖的语种对象清单
本数据库详细解析了以下 12 种母语背景学习者的痛点与教学对策：
* **印欧语系**：英语、西班牙语、法语、俄语、德语、葡萄牙语、印地语
* **汉字文化圈（受汉语影响较深）**：日语、韩语、越南语
* **其他特色语系**：阿拉伯语（书写系统差异）、泰语（声调系统差异）
